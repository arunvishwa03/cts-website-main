DMS Softwares

OpenDocMan

OpenDocMan is one of the best open-source document management systems. It is not only free but also web-based so that it can be accessed from anywhere rather than just an office computer. It supports multiple operating systems and is suitable for both individual and departmental workflow.

Kimios

Kimios is highly optimised to help firms with heavy document flows to stay organised. This is very useful for departments like human resources and accounts who use plenty of documentation every day. It is designed to fit all web standards and functions effectively across all platforms.

Alfresco Community Edition

Alfresco Community Edition is the best choice for those who have the capability of self-support. Written mainly in JAVA and PostgreSQL for storing databases, this opensource tool is not only free but also very widely used which makes it one of the top choices.

LogicalDOC Community Edition

LogicalDOC Community Edition is the open-source version of LogicalDOC, similar to Alfresco Community Edition. This version comes free of charge and has the same features as the LogicalDOC version with a range of features and benefits. It comes with multilingual text indexing and dropbox integration making it a very popular choice among organisations.

OpenKM Community Edition

OpenKM Community addition is the free version of OpenKM. It can be accessed via the web making remote access easy. Developed on the Java platform, it can be accessed via the phone also. It is a great option for firms with few users and non-critical data.
